Material Setup Guide

Learn how to correctly create and import materials using APE.

1. Preparing Your Textures

  • Export textures as PNG or TGA
  • Make sure normal maps end with _n
  • Keep roughness/metallic maps clean and high‑contrast

2. Creating a Material in APE

  • Open APE and create a new material
  • Set the correct shader (usually mc_sw4 or mc_sw4_add)
  • Assign your color, normal, and other maps

3. Common Issues

  • Material appears shiny → roughness map missing
  • Material looks flat → normal map not linked
  • Material is black → texture path incorrect
